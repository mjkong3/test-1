### 글꼴 변경
- 굵게 : 텍스트를 ** 또는 __ 로 감싸기

- 기울림체 : 텍스트를 * 또는 _ 로 감싸기

- 굵은 기울림체 : 텍스트를 *** 또는 ___ 로 감싸기

- 취소선 : 텍스트를 ~~ 로 감싸기

### 소스코드 삽입
백틱(`) 사이에 작성. 백틱 뒤에 코드 언어를 작성해줘요

여러줄 소스코드 
```java

  int result = x+y;
  return result;
};
```
